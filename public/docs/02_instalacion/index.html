Instalando en Windows con XAMPP

XAMPP es una solución sencilla para tener Apache, PHP, y MySQL en Windows. Sigue estos pasos para instalarlo:

Descargar XAMPP Ve al sitio web oficial de XAMPP https://www.apachefriends.org y descarga la versión que incluye PHP.

Instalar XAMPP Ejecuta el instalador y sigue las instrucciones. Asegúrate de seleccionar Apache y PHP durante la instalación.

Iniciar los servicios : Abre el panel de control de XAMPP y asegúrate de iniciar Apache.

Comprobar instalación Abre tu navegador y visita http://localhost/. Deberías ver la página de bienvenida de XAMPP.

Creación de docker

  • Es esta la solución que vamos a utilizar.
  • Para ello crearemos un docker-compose.yaml , que nos levante un servicio con php y apache2.
  • Por si queremos realizar modificaciones sobre la imagen original, la imagen la vamos a crear a patir de un Dockerfile
  • Cada proyecto que creemos o bien tendremos esta estructura, o bien los crearemos todos a partir de la carpeta compartida

Info

  • Creamos el servicio (contenedor ) web
  • A partir del Dockerfile que tenemos en la misma carpeta
  • Proyectamos el puerto 8800 para acceder a él
  • Mapeamos la carpeta ./../app para escribir nuestros programas

FROM php:8.3-apache
ENV DEBIAN_FRONTEND=noninteractive
RUN apt update
# Usa la imagen base de PHP con Apache

# Añadir o modificar la configuración de Apache para listar directorios si no hay index
RUN echo '<Directory /var/www/html>' >> /etc/apache2/apache2.conf
RUN echo 'Options +Indexes' >> /etc/apache2/apache2.conf
RUN echo '</Directory>' >> /etc/apache2/apache2.conf

# Exponer el puerto 80
EXPOSE 80
