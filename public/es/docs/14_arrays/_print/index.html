Función array_map

array_map
La función array_map aplica una función callback a cada elemento de uno o más arrays y devuelve un array con los resultados.

Sintaxis: 

1

array_map(callable $callback, array $array, array ...$arrays): array

Ejemplo: 

1
2
3
4
5
6

function sumarUno($n) {
return $n + 1;
}

$resultado = array_map('sumarUno', [1, 2, 3]);
// Resultado: [2, 3, 4]

Usamos un callback para transformar los elementos. Un callback representa un tipo de dato que corresponde al código de una función, es el tipo del parámetro (en lugar de ser de otro tipo el parámetro, como entero, real, …)

Función array_fill

array_fill
La función `array_fill` llena un array con valores específicos, comenzando desde un índice dado.

Sintaxis: 

1

array_fill(int $start_index, int $count, mixed $value): array

Ejemplo: 

1
2

$resultado = array_fill(0, 5, 'PHP');
// Resultado: ['PHP', 'PHP', 'PHP', 'PHP', 'PHP']

Se genera un array con un valor repetido n veces.

Función range

range
La función range crea un array con una secuencia de valores entre un inicio y un final, opcionalmente con un paso.

Sintaxis: 

1

range(mixed $start, mixed $end, int|float $step = 1): array

Ejemplo: 

1
2

$resultado = range(1, 10, 2);
// Resultado: [1, 3, 5, 7, 9]

Genera un array secuencial entre valores.
